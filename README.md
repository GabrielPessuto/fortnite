# 🎮 FNCS Friends

Site estilo Fortnite com fundo animado e tabela de amigos.

---

## 💜 Sobre o projeto
Este projeto é uma página web com tema Fortnite, contendo:
- Fundo animado com partículas neon
- Interface estilo gamer
- Lista de amigos

---

## 🚀 Tecnologias usadas
- HTML5
- CSS3
- JavaScript
- tsParticles
